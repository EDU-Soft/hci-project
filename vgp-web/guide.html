User Guide

It's finally here. We can now explore the beautiful world of geometry using only voice! But how do we do that? Lets see below. Let's get started by setting our whiteboard title:

set title all possible shapes

Since we're still in the beginning let's also define the drawing properties that we are going to use:

set line color blue

set fill color white

set size 3

set opacity 0

Now that we are 'prepared', lets draw! We'll start with something simple, a point:

draw point A 10 12

... and continue with something twice more complex:

draw line MN 5 cmd_del 7 cmd_del 20 cmd_del 30

Easy peasy! Let's draw more complex shapes! Triangles:

all triangle help cmds

Quadrilaterals:

all quadri help cmds

Circular shapes:

circle cmd help

By trying all the commands above we ended up with a full board. How do we clear it? Simple:

clear board cmd help

Oups! We forgot to save a copy of the drawing or share it? What do we do? 1. Let's undo: undo cmd help 2. Download: download cmd help 3. Share on dropbox: share on dropbox cmd help
